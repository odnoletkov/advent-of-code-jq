# advent-of-code-jq

Solving [Advent of Code](https://adventofcode.com) with [jq](https://github.com/stedolan/jq)

## Run with input file

    $ jq -nRf 2020/01-1.jq your_input.txt

## Or fetch input automatically (macOS)

    $ make save-session
    ... paste 'session' cookie from browser
    $ make 2020/01-1
